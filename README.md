# きょうやる

今日やることだけに集中するための、シンプルで気持ちいいToDoアプリです。
インストール不要で、スマホやPCのブラウザからそのまま使えます。ホーム画面に追加すると、アプリのように起動できます。

## 公開URL

- 使う: https://omicreate.github.io/kyou-yaru/
- リポジトリ: https://github.com/omicreate/kyou-yaru

## 主な機能

- タスクの追加 / 完了 / 編集 / 削除（編集はタスクをタップ → ボトムシート）
- 「じぶん／しごと」の切り替え。ストリークと日々の実績は両方合算（分けると土日に途切れるため意図的に共通）
- 音声入力。🎤タップ→話す→入力欄に反映→＋で追加。ブラウザが音声認識に対応していない環境ではボタン自体が非表示（その場合もOSキーボードの音声入力は利用可）
- 期限・優先度（低/中/高）・タグ・検索・フィルタ
- 「きょう」トグルで今日ビューに送る（期限が今日のタスクも自動で今日ビューに出る）
- ドラッグ並べ替え（タッチは長押しで開始、マウスは ⠿ ハンドル）
- ストリーク（1日1件以上完了で継続、次のマイルストーンまでの日数を表示）・継続カレンダー（使い始めた週から表示、最大直近20週の草グラフ、タップで日別詳細）・直近14日グラフ・タグ別/スペース別集計
- ルーチン（繰り返しタスク）: 曜日を指定して毎日/平日/カスタムで繰り返し。完了すると次の該当曜日にまた出現し、持ち越しは発生しない
- ダークモード（自動 / ライト / ダーク）
- 今日ぜんぶ完了で「済」スタンプ + 紙吹雪
- オフライン利用に備えたPWA対応

## 設計方針：機能を絞ることが売り

多機能・AI連携をうたうToDoアプリと機能の数で張り合いません。「今日やること」だけに集中できる状態を保つため、次のことはあえてやりません。

- カレンダー同期・外部サービス連携
- AIによる自動タスク生成・要約・提案
- チーム機能・共有機能

追加するのは「今日に集中する」という軸を研ぎ澄ますものだけです。機能が少ないのは手抜きではなく方針です。

## 使い方

1. 公開URLをブラウザで開きます。
2. 画面下の入力欄からタスクを追加します（🎤で音声入力も可能）。
3. タスク左の丸をタップするとチェックが付き、完了として記録されます（タスク本体をタップすると編集画面が開きます）。
4. `すべて` タブで期限・優先度・タグを設定したり、検索・並べ替えができます。
5. `きろく` タブで継続日数や完了実績を確認します。

## データ/プライバシー

- アカウント登録は不要です。
- 入力データは端末内のブラウザ（localStorage）に保存され、開発者や外部サーバーへは送信されません。アクセス解析も行っていません。
- 音声入力は、ブラウザに搭載された音声認識機能（Web Speech API）を利用します。音声の認識処理はブラウザ提供元（Chromeの場合はGoogle）側で行われる場合があり、これはomicreateの管理下にはありません。使いたくない場合は、音声入力ボタンを使わずキーボード入力のみで利用できます。
- 詳細は [PRIVACY.md](PRIVACY.md) を確認してください。

## 技術構成

- 実行方式: 静的HTML / CSS / JavaScript（単一ファイル、ビルド不要）
- 主なファイル: `index.html`
- PWA関連: `manifest.webmanifest`, `sw.js`, `icons/`

## 確認方法

```sh
npm run check:public-info
```

ビルドは不要です。ブラウザで公開URLを開き、以下を確認します。

- タスクの追加 / 完了 / 編集 / 削除がリロード後も保持される
- `きろく` タブでストリークと継続カレンダーが表示される
- ホーム画面に追加した状態でオフラインでも起動する

## 更新時の注意

アプリを更新したら `sw.js` の `CACHE_NAME` のバージョンを上げること（例: `kyou-yaru-v1.0.1`）。

# セキュリティ方針

最終更新: 2026-07-07

## 対象

この文書は、きょうやるの公開Webアプリと、このリポジトリ内のコード、ドキュメントを対象にします。

## 基本方針

- 入力データは端末内保存を基本とする
- 外部APIへタスクデータを自動送信しない（音声入力のブラウザ標準機能を除く）
- 個人名を含む実データはGit管理しない
- HTML表示ではユーザー入力をエスケープする

## 脆弱性を見つけた場合

現時点では個人開発・テスト公開のため、GitHub Issuesで詳細な個人情報や悪用手順を公開しないでください。

連絡時は、以下だけを簡潔に共有してください。

- どの画面で起きたか
- どのブラウザ、端末で起きたか
- 期待した動作と実際の動作
- 個人情報を含まない再現手順

## 公開してはいけない情報

- APIキー、トークン、Cookie、認証情報
- 個人端末の絶対パス、作業フォルダ名、端末固有の一時フォルダ
- 非公開の起動URL、検証URL、作業用URL
- 個人端末のバックアップファイル

## 公開前チェック

公開前に次を実行します。

```bash
npm run check:public-info
```

このチェックは、公開文書やコードに個人環境由来の情報が混ざっていないかを確認するためのものです。

## 依存関係

本アプリは静的HTML/CSS/JavaScriptのみで動作し、外部ライブラリへの依存はありません。

## 免責

このアプリはタスク管理を補助するものです。完全なセキュリティやデータ保持を保証するものではありません。
